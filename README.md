# ATC DNS Presentation

Single-file Reveal.js presentation for the feasibility study on physics-informed DNS of particle suspensions in the Archimedes Tube Crystallizer (ATC). The deck covers motivation, ATC setup, governing equations and numerical methods, hard-contact modeling, DNS vs. LPT comparisons, experimental validation, quantitative regime metrics, results, conclusions, and outlook. Figures are currently placeholders in the HTML.

## Run

Open the file directly in a browser:

```bash
open presentation.html
```

If `open` is not available, use:

```bash
xdg-open presentation.html
```

You can also serve it locally:

```bash
python3 -m http.server 8000
```

Then navigate to:

```text
http://localhost:8000/presentation.html
```
